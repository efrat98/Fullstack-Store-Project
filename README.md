# Fullstack Store Project

A comprehensive e-commerce application featuring a JavaScript frontend, a Python-based backend API, and a SQL Server database.

## Project Structure
- **Frontend:** A dynamic user interface built with HTML, CSS, and Vanilla JavaScript.
- **Backend:** A RESTful API built with Python to manage store data and business logic.
- **Database:** SQL Server (Relational database for products, users, and orders).

## Technologies
- **Client:** JavaScript, HTML5, CSS3
- **Server:** Python
- **Database:** Microsoft SQL Server
- **Data:** JSON/API integration

## Features
- Dynamic product display
- Shopping cart functionality
- Backend API for managing store items
- SQL Database for persistent storage
